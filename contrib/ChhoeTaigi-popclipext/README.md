# 查台語（找台語） PopClip 擴充功能

這是一個超輕量的 PopClip 插件，讓你在 macOS 上選取任何文字後，
一鍵查詢「找台語」（[chhoe.taigi.info](https://chhoe.taigi.info)）網站。

---

## ✨ 功能說明
- 選取任意華語或台語文字
- 點擊 PopClip 上的「查台語」按鈕
- 自動在預設瀏覽器開啟查詢結果

---

## 🧩 安裝方式
1. 下載本資料夾或 `.popclipext` 封裝檔
2. 雙擊安裝（PopClip 會自動匯入）
3. 試著在任何地方選字 → 彈出按鈕 → 點「查台語」
